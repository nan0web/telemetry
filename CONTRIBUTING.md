# Contributing

We welcome contributions! Please follow our coding style and ensure all tests pass.

## Workflow

1. Fork the repo.
2. Create your feature branch.
3. Commit your changes.
4. Push to the branch.
5. Create a new Pull Request.

## Coding Style

We follow the NAN0 coding style:

- Tabs for indentation.
- No semicolons.
- JSDoc for types.
- TDD with node:test.
